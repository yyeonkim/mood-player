# Mood Player 🎵
2021 학교 개인 프로젝트로 만든 **<키워드 맞춤 노래 제공 어플>**
- 기분 키워드를 2개 선택하면, 그에 맞는 플레이리스트를 제공합니다.
- 이미지와 플레이리스트는 랜덤 제공이 아니라, 미리 만든 object를 줍니다.
